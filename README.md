# Invoice Management System

A full-stack Invoice Management System built using React.js, Node.js, Express.js, and MySQL.

## Features
- User Authentication
- Customer Management
- Invoice Creation
- CRUD Operations
- Database Integration
- Responsive UI

## Technologies Used
- React.js
- Node.js
- Express.js
- MySQL
- Git & GitHub
